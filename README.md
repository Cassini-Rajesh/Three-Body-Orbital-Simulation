# Three-Body Orbital Simulation (CR3BP)

## Overview
Simulated spacecraft trajectories in the Earth-Moon system using 
CR3BP dynamics and rotating frame analysis in MATLAB.

## Tools & Methods
- MATLAB
- CR3BP dynamics and rotating frame analysis
- Eigenvalue decomposition for stability analysis
- ODE45 numerical integration

## Key Results
- Computed Lyapunov orbits around L2 and L4 Lagrange points
- Evaluated stability via eigenvalue decomposition
- Compared nonlinear vs linearly perturbed trajectories in 
  rotating and inertial frames
